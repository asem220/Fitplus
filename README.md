# FitPlus

## Описание проекта

**FitPlus** — это SPA-приложение для формирования персонального плана тренировок и питания с использованием ИИ.

Пользователь проходит анкетирование, после чего система:

* формирует индивидуальный фитнес-план;
* даёт рекомендации по питанию;
* позволяет отслеживать прогресс;
* предоставляет чат с ИИ для корректировки плана.

Проект построен как fullstack-приложение с авторизацией, базой данных и AI-интеграцией.

---

## Основной функционал

### 👤 Аутентификация

* регистрация и вход (JWT);
* защита маршрутов;
* удаление аккаунта с полным удалением данных.

### 🧠 Генерация плана (ИИ)

* создание персонального плана тренировок;
* рекомендации по питанию;
* советы по отслеживанию прогресса;
* ограничения и безопасность.

### 💬 AI-чат

* пользователь может уточнить или изменить план;
* взаимодействие в формате диалога;
* ответы адаптируются под профиль пользователя.

### 📊 Отслеживание прогресса

* запись веса;
* отметка выполнения тренировок;
* заметки по питанию и самочувствию;
* история последних записей.

### 🧪 Опрос (анкетирование)

* сбор данных о пользователе;
* формирование профиля для ИИ;
* сохранение результатов.

### 🖼 Профиль

* отображение аватара;
* история активности;
* текущий AI-план.

## API

### POST `/api/auth/register`

Регистрация пользователя

### POST `/api/auth/login`

Авторизация

### GET `/api/auth/profile`

Получение профиля (JWT)

### DELETE `/api/auth/delete`

Удаление аккаунта (JWT)

---

### POST `/api/ai/plan`

Генерация персонального плана через ИИ

---

### POST `/api/user/progress`

Добавление записи прогресса:

```json
{
  "weight": number,
  "workoutCompleted": boolean,
  "nutritionNote": string,
  "note": string
}
```

---

### POST `/api/auth/upload-avatar`

Загрузка аватара

---

### POST `/api/quiz/save`

Сохранение результатов опроса

## Как запустить локально

```bash
git clone <repo_url>
cd fitplus
npm install
```

Создать `.env`:

```env
MONGODB_URI=mongodb://localhost:27017/testify
JWT_SECRET=your_secret
GROQ_API_KEY=your_groq_key
```

Запуск:

```bash
npm run dev
```

Открыть:

```
http://localhost:3000
```
